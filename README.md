# Zelda-Map

**塞尔达传说：旷野之息 交互式地图**

一个基于 Leaflet 构建的《塞尔达传说：旷野之息》(The Legend of Zelda: Breath of the Wild) 交互式地图，旨在为玩家提供直观、便捷的游戏探索辅助工具。

## 项目特性

- 🗺️ 高清游戏地图可视化展示
- 🌍 支持地图缩放、平移等基础交互操作
- 🛠️ 基于 React + Vite 构建，兼顾开发效率与运行性能

## 技术栈
技术	|说明
--------|-----
React	| 前端 UI 框架，构建组件化界面
Vite	| 构建工具，提供快速热更新 (HMR) 与优化构建能力
Leaflet	| 开源地图库，实现地图渲染与交互功能
Material UI	| 基于 Material Design 的 React 组件库，快速构建美观的 UI 界面

## 快速开始

```bash
npm install
npm run dev
```
